
# 1.2 运行Hello World

我们就在默认生成的 `App.java` 文件中写上程序员入门必备的 `Hello World`。

```java
package com.bladejava.firstapp;

import com.blade.Blade;

/**
 * Hello world!
 *
 */
public class App {
	public static void main(String[] args) {
		Blade.me().get("/",	(request, response) -> response.text("Hello World")).start();
	}
}
```
